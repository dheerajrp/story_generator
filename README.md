# story_line_generator
Story Line Generator

- Make sure python is installed in your system.

- Clone the repo - 
```bash
git clone https://github.com/dheerajrp/story_generator.git
```

- Navigate to the repo directory
```bash
cd story_generator
```

- Run the file -
```bash
python story_teller.py
```
